Here's everything an advanced productivity website should have:

---

**📋 Task Management**

- Quick capture (add tasks in under 2 seconds)
- Subtasks, dependencies, and due dates
- Priority levels and labels/tags
- Recurring tasks
- Kanban, list, and calendar views

**⏱ Focus & Time**

- Pomodoro timer with work/short break/long break modes
- Session history and focus logs
- Time tracking per task or project
- Daily time budget

**✅ Habits**

- Daily habit tracker with streaks
- 30-day dot grid view (like GitHub's graph)
- Completion rate stats per habit
- Multiple frequencies (daily, weekly, custom)

**📝 Notes**

- Rich markdown note editor
- Tagging and full-text search
- Link notes to tasks or projects
- Pinned and archived notes

**📊 Dashboard & Analytics**

- Daily/weekly task completion stats
- Focus time heatmap
- Habit streak overview
- Productivity score over time

**📅 Planning**

- Weekly review mode
- Time blocking on a calendar
- Goals broken into milestones
- Daily "top 3" priorities

**⚙️ Power Features**

- Keyboard shortcuts for everything
- Command palette (like Notion's `/` or VS Code's `Ctrl+P`)
- Dark/light mode
- Offline support
- Data export (CSV, JSON)
- Customizable widgets on the dashboard

---

The secret sauce is making all of these feel **connected** — e.g. a task links to a note, which links to a focus session, which shows up in your weekly review. That integration is what separates great productivity apps from a pile of disconnected tools.
